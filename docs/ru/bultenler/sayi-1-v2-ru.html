Профиль

Мейнстрим академик. Прогнозы согласованы с консенсусом.

Ключевые наблюдения

  • USD/TRY «диапазон 45-54 ТЛ» → полностью соответствует Опросу Ожиданий ЦБ ТР (51.57 на конец года, 54.69 через 12 мес.)
  • Прогноз дефицита текущего счёта 2023 г. $45 млрд → факт TÜİK $45.4 млрд (якорная валидация)
  • Прогноз снижения премии CDS → совпадает с фактическим движением
Ценность для Tradia: Валидатор консенсуса. Когда сигналы противоречат опросу ЦБ ТР, Махфи служит контрольной точкой.
Пробел: НЕ даёт прогнозов по жилью отдельно (только 1/100 постов). Махфи даёт макро-рамку; для прямого сигнала по недвижимости нужны другие источники.

Методология

Формула последовательности

Балл = (совпадение_тем × 0.4 + числовая_плотность × 0.4 + верификация_tradia × 0.2) × 100
  • совпадение_тем: Сходство Жаккара YouTube и письменного контента по каждому общему месяцу (13 ключевых экономических тем)
  • числовая_плотность: Доля полного текста × 0.4 + 0.4 (фиксированная база)
  • верификация_tradia: Согласование конкретных прогнозов аналитика с КФЭ ЦБ ТР + Опросом Ожиданий + TÜİK (0-1)

  • ▸ Верификация: данные ЦБ ТР (2026-05), TÜİK, новостной пул Tradia
