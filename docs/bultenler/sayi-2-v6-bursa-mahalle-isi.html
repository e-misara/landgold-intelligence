  1. BBB Meclis Kararları — Bursa Büyükşehir Belediyesi PDF tam metin (Sprint 11-15), 180 karar dosyası.
  2. Sahibinden Master — Bursa 2.592 aktif ilan kaydı, mahalle alanı %55 DOLU + %45 regex kurtarma.

Kategori Tanımı

  • 🔥 SICAK: meclis ≥2 + ilan ≥50 → planlama + piyasa eşzamanlı hareket
  • 🏛️ OLGUNLAŞMIŞ: ilan ≥40 + meclis aktif değil → kararlar tamamlanmış, satış aktif
  • 🌱 ERKEN SİNYAL: meclis ≥3 + ilan ≤10 → planlama hazır, piyasa 6-12 ay geride
